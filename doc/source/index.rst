Welcome to lmrob's documentation!
=================================

The lmrob package is a translation of the lmrob functions of the robustbase
package of R code. The included functions are the following:

* mad
* tauHuber
* huberM
* wgtHighMedian
* sumU
* lmrob
* lmrob_control
* lmrob_fit
* lmrob_S
* lmrob_D_fit
* lmrob_M_fit
* lmrob_M_S
* lmrob_lar
